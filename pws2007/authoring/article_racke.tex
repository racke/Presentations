\subsection{Grundregeln}

Prinzipiell kann man sich recht frei entfalten. Die Beiträge sind
\verb/\section/s eines Gesamtdokumentes der KOMA-Klasse {\em
scrartcl}. Damit sich alle Beiträge zusammenfügen, hier die
wichtigsten Hinweise.

\begin{itemize}

\item Der Beitrag ist eine Datei {\em article\_AUTORKÜRZEL.tex}.

\item Der Beitrag beginnt mit \verb/\section/.

\item Es darf {\bf kein} \verb/\usepackage/ verwendet
werden\footnote{Sollte wirklich ein unverzichtbares Paket fehlen,
erweitern wir unsere Perlworkshop-Style-Datei.}.

\item Alle verfügbaren Packages sind der Datei {\em perlws.sty}
entnehmbar. 

\item Der Beitrag muß im Rahmen der mitgelieferten, unveränderten
Datei {\em main.tex} funktionieren, welche den Artikel importiert
und strukturell die letztendliche \LaTeX-Umgebung simuliert.

\item Labels für Verweise, Bibliographie-Einträge, müssen mit
einem eindeutigen Präfix versehen werden, z.B. mit einem
Autorenkürzel. Im vorliegenden Beispielartikel wird durchgängig
\verb/racke:/ verwendet.

\item Zusätzliche Dateien (z.B. Grafiken) müssen in genau einem
getrennten Unterverzeichnis abgelegt werden, z.B. bestehend aus dem
Autorenkürzel. Im vorliegenden Beispielartikel wurde \verb/racke/
verwendet.

\end{itemize}

\noindent
Der vorliegende Artikel dient als Beispiel für einen korrekten Artikel
und demonstriert wichtige \LaTeX-Elemente.

\subsection{Main rules}

Generally feel free. The contributions are \verb/\section/s of a main
document of the KOMA class {\em scrartcl}. This class is more german
language centric, especially regarding layout. Anyway it should work
for you, too.

For nicely fitting all contributions together, here are the most
important hints.

\begin{itemize}

\item The contribution is a file {\em article\_AUTHORMNEMONIC.tex}.

\item The contribution starts with \verb/\section/.

\item You {\bf must not} use \verb/\usepackage/. \footnote{In case
you really, really need an additional package, write us and we try to
extend our perlworkshop style file.}

\item All available packages can be seen in the file {\em perlws.sty}.

\item The contribution must work within the context of the provided file {\em main.tex}, which imports your article and simulates our final \LaTeX environment.

\item Labels for cross references, bibliography entries, etc. must me
used with unique prefix, i.e. the AUTHORMNEMONIC. In the current
example I use \verb/racke:/.

\item Additional files (e.g., images) must be provided within a
separate sub directory, e.g., named like your AUTHORMNEMONIC.  In the
current example I use sub directory \verb/racke/.

\end{itemize}

\noindent
The current article is an example for a correct contribution and
demonstrates some important \LaTeX features.
